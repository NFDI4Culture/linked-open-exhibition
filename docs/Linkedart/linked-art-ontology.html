Linked Art Ontology — Things (Classes) Reference

Based on the Linked Art Data Model 1.0 — an application profile of CIDOC-CRM 7.1.3 for describing cultural heritage resources.

Overview

Linked Art defines a streamlined subset of CIDOC-CRM classes organised into core entities (the primary things you describe), event/activity classes (things that happen), and supporting classes (structures embedded within descriptions). The ontology is serialised as JSON-LD 1.1.
